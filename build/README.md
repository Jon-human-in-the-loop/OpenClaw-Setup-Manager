# Build Assets

Place your app icons here before packaging:

| File | Size | Platform |
|------|------|----------|
| `icon.ico` | 256x256 | Windows |
| `icon.icns` | 512x512 | macOS |
| `icons/16x16.png` | 16x16 | Linux |
| `icons/32x32.png` | 32x32 | Linux |
| `icons/48x48.png` | 48x48 | Linux |
| `icons/64x64.png` | 64x64 | Linux |
| `icons/128x128.png` | 128x128 | Linux |
| `icons/256x256.png` | 256x256 | Linux |
| `icons/512x512.png` | 512x512 | Linux |

Use `icon.svg` as the source to generate all formats.
Tool: https://www.iconconverter.com/ or https://cloudconvert.com/
